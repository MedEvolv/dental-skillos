Dental SkillOS - Mirror-Guided Target Tracing

Indirect Vision Practice Task

Instructions:

  • Position mirror to view the path indirectly
  • Keep eyes on mirror reflection, NOT on the target directly
  • Use probe/stylus to trace the black path from START (green) to END (red)
  • Maintain mirror stability throughout the movement
  • Stay within the path boundaries
  • Complete in one continuous motion if possible
