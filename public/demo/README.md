# Demo Videos

This folder contains the demo videos for the Dental SkillOS prototype.

## Required Videos

### attempt1.mp4 (Poor Performance)
- **Duration**: ~200 seconds (3 min 20 sec)
- **Scenario**: First attempt, struggling with mirror control
- **Key behaviors**: Frequent mirror repositioning, field loss, direct vision cheating, target misses, shaky movements

### attempt2.mp4 (Improved Performance)
- **Duration**: ~168 seconds (2 min 48 sec)
- **Scenario**: After practice, showing improvement
- **Key behaviors**: Stable mirror, smooth movements, fewer errors, better posture

### attempt3.mp4 (Fatigue Case)
- **Duration**: ~245 seconds (4 min 5 sec)
- **Scenario**: Late-session performance degradation
- **Key behaviors**: Starts okay, progressively worsens, posture drift, hesitation, increased errors in final third

## Recording Specifications

- **Resolution**: 1920x1080 (1080p)
- **Frame rate**: 30fps
- **Codec**: H.264
- **Format**: MP4
- **Audio**: Clear voice stating attempt number

## How to Create

1. Set up workspace per `/RECORDING_PROTOCOL.md`
2. Use the same actor for all 3 videos
3. Film all 3 in one session for consistency
4. Follow the specific action scripts in the protocol
5. Export and place files here with exact filenames

## Note

These are simulated videos for demo purposes. The AI analysis in this prototype uses pre-defined event data in `app/data/sampleData.ts` that corresponds to these video durations and scenarios.
