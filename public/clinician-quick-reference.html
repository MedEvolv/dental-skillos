Clinical Quick Reference

Smartphone-Based Miniaturized Target Protocol

🔬 Why Miniaturized Digital Targets?

Clinical dentistry operates at submillimeter precision (0.1-0.5mm). Traditional A4 printed targets create a mismatch between training scale and clinical reality. Our 2cm digital targets replicate actual working field size while enabling precise measurement of micro-movements via macro recording.

Equipment Required

Target Device Smartphone/tablet with 5"+ screen
Recording Device Smartphone with macro mode, 1080p/60fps
Tripod/Stand For recording phone (15-20cm height)
Lighting Ring light or dual LED sources
Instruments Dental mirror (#5) + probe/explorer
Protection Tempered glass screen protector

Setup Configuration

Target Specifications

Display Settings

  • Active Area: 2cm × 2cm (scalable: 1.5cm - 2.6cm)
  • Path Width: 0.5mm - 2mm (difficulty-adjusted)
  • Resolution: Screen-native (typically 300+ PPI)
  • Contrast: Black path on white background
  • Markers: Green start dot, Red end dot

Difficulty Progression

Beginner: 2mm width Intermediate: 1mm width Advanced: 0.5mm width Expert: 0.3mm width

Recording Protocol

Pre-Recording Checklist

Target phone brightness at 80-100%
Screen protector clean, no scratches
Recording phone locked at 15-20cm distance
Macro mode enabled, focus locked on target
Lighting balanced (no shadows on screen)
Frame includes entire target + both hands
Attempt number announced at start

Clinical Applications

Transferable Skills

  • Restorative: Mirror-guided cavity prep, matrix placement
  • Endodontics: Canal location under rubber dam
  • Periodontics: Subgingival scaling visualization
  • Prosthodontics: Crown prep margin visibility
  • Surgery: Third molar site visualization

Workflow Summary

  1. Assign: Student receives target via digital-target-display.html
  2. Setup: Position equipment per protocol
  3. Practice: Student traces target using mirror only
  4. Record: Macro video captures entire session
  5. Upload: Video submitted to system
  6. Analyze: AI extracts metrics (2-5 minutes)
  7. Review: Faculty sees video + insights
  8. Feedback: Structured report to student

⚠️ Important Notes

  • Replace screen protector when scratched (affects tracking accuracy)
  • Calibrate target size if using non-standard screen density
  • Ensure consistent lighting across sessions for valid comparisons
  • Student must maintain indirect vision - direct looking = invalid attempt
